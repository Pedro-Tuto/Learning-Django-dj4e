Um Header

Aqui colocamos algumas regras uma tag de <style> , e elas sao aplicadas a partes do documento que atendam aos seletores.

Depois de um tempo, acabamos com muitas regras CSS para colocar na area <head> de cada documento. Uma maneira mais simples de incluir um numero grande de regras CSS e coloca-los em um arquivo separado e incluir esse arquivo em cada uma das paginas.
